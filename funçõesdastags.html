Definições dos elementos e atributos da tabela.

  • TABLE-é usada para definir tabelas. As tabelas são constituídas por linhas e por células
  • TH-As tabelas podem ter cabeçalhos. Para isso usa-se a etiqueta th que significa table header.
  • TR-A etiqueta tr que significa table row é usada para definir uma linha
  • TD-A etiqueta td que significa table data é usada para definir uma célula.
  • THEAD-O elemento thead é utilizado para delimitar a área de cabeçalho na tabela
  • TBODY-tag é usada para formar uma combinação de conteúdo do corpo HTML.
  • TFOOT-A tag tfoot é utilizada para definir conteúdos de rodapé dentro de uma tabela e juntá-lo ao corpo e ao cabeçalho
  • ALIGN- Este atributo enumerado indica como a tabela deve ser alinhada considerando o conteúdo do documento.
  • BORDER-A presença do atributo border indica a espessura da borda que deve ser desenhada em torno de cada célula da tabela.
  • CAPTION-O Elemento caption representa o título de uma tabela
  • ROWSPAN-permite criar uma célula que ocupa várias linhas.
  • COLSPAN- permite criar uma célula que ocupa várias colunas;
  • WIDTH-O atributo width especifica a largura da tabela.
